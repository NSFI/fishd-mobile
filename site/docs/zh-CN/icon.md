## API

| 参数    | 说明                                 | 类型               | 默认值    |
| ------- | ------------------------------------ | ------------------ | --------- |
| type    | 内置 icon 名称                       | `string`           |
| dot     | 是否显示图标右上角小红点             | `boolean`          | `false`   |
| badge   | 图标右上角徽标的内容                 | `string`           | -         |
| color   | 图标颜色                             | `string`           | `inherit` |
| size    | 图标大小，如 20px 2em，默认单位为 px | `number \| string` | `inherit` |
| onClick | 点击图标时触发                       | `event: Event`     | -         |
